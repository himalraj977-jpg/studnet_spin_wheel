# 🎡 Student Spin Wheel Web Application

## 📌 Project Overview

The Student Spin Wheel is a simple interactive web application that randomly selects a student or option from a spinning wheel.  
This project demonstrates front-end web development concepts using HTML, CSS, and JavaScript.

It was developed as part of a software/application development assignment.

---

## 🌐 Live Website

The application is deployed using GitHub Pages and can be accessed here:

🔗 https://himalraj977-jpg.github.io/studnet_spin_wheel/

---

## 🚀 Features

- Interactive spinning wheel
- Random selection logic
- Smooth rotation animation
- Clean and simple user interface
- Lightweight and fast (no external libraries used)
- Fully browser-based (no installation required)

---

## 🛠 Technologies Used

- HTML5 – Structure of the application
- CSS3 – Styling and animation
- JavaScript (Vanilla JS) – Logic and interactivity
- GitHub Pages – Deployment
